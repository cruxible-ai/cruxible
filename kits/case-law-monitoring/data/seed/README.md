# Case-law corpus seed (two-act)

Act one: real public opinions from the Chevron-deference cluster
(CourtListener metadata; opinions are public domain, holding summaries are
authored here) plus the synthetic firm layer. Act two: the update fixture
that brings in the overruling opinion, consumed by refresh_corpus when
offline. Seed CSVs land with the provider implementation; this bundle is
digest-pinned in config.yaml.
